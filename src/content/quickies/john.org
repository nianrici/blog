#+description: Artículo sobre John
John the Ripper es una herramienta de software de seguridad que se
utiliza para intentar descifrar contraseñas. Se utiliza a menudo para
probar la seguridad de las contraseñas de un sistema, ya que puede
intentar "adivinar" las contraseñas utilizando diferentes técnicas, como
usando diccionarios o por fuerza bruta. Si John the Ripper es capaz de
descifrar una contraseña, significa que la contraseña no es lo
suficientemente segura y debería ser cambiada por una más fuerte.
