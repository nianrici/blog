# Niko's Blog

A personal blog built with [Astro](https://astro.build) and [Org Mode](https://orgmode.org).

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Content

Blog posts are written in Org Mode and located in `src/content/blog/`.
Quickies (short notes) are in `src/content/quickies/`.
